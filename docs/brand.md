# Brand And Logo

溯知的英文名是 `TraceMind`，含义是：沿着证据的踪迹，形成可以被追溯的理解。

## Logo

主 logo 位于：

- [assets/tracemind-logo.svg](../assets/tracemind-logo.svg)

应用图标位于：

- [frontend/public/favicon.svg](../frontend/public/favicon.svg)

两个文件都是透明背景 SVG。

## 图形含义

Logo 由三层隐喻组成：

- 左侧纵向节点表示研究的原始时间线：问题、证据、判断不断沉积。
- 两条分叉路径表示研究会从一个线索分裂成不同问题和方向。
- 中间的「证据之眼」表示溯知的核心原则：任何判断都要能回看来源。

## 风格原则

- 极简，不堆叠图标。
- 高对比，适合浅色文档和 GitHub README。
- 少量强调色：黑色代表判断，青绿色代表证据，蓝色代表推理分支，金色代表当前焦点。
- 不使用背景色，让 logo 能贴在不同页面和卡片上。

## 推荐用法

在文档或产品页中使用：

```html
<img src="../assets/tracemind-logo.svg" alt="TraceMind logo" width="520">
```

不要拉伸变形，不要在低对比背景上使用，不要额外添加阴影或渐变。
