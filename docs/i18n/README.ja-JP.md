# TraceMind

TraceMind は、本格的な研究読解と調査のための AI 研究ワークベンチです。論文発見、証拠抽出、研究ノード、判断の記述、根拠付きの対話を、追跡可能な一つの流れにまとめます。

## 解決する課題

- 研究資料が論文データベース、PDF、メモ、チャット履歴に分散している。
- AI の回答は自然でも、根拠の道筋が見えにくい。
- 研究テーマは継続的に変化するが、通常のチャットには長期的なテーマ記憶がない。

TraceMind は論文、図、数式、引用、ノード、会話を同じ文脈に置き、各主張の背後にある証拠を確認できるようにします。

## 使い方

1. バックエンドとフロントエンドを起動します。
2. Settings で言語モデルと任意の視覚モデルを設定します。
3. 研究テーマを作成または開きます。
4. 論文探索を実行し、候補を確認します。
5. ノードページで証拠、図、数式、引用を読みます。
6. ワークベンチで根拠付きの質問を続け、成果物をエクスポートします。

## 仕組み

TraceMind は React + Vite フロントエンド、Express + Prisma バックエンド、Omni モデルゲートウェイ、生成データスナップショットで構成されています。バックエンドは学術検索源を集約し、PDF から証拠を抽出し、研究ノードを構築し、モデル呼び出しを設定可能なゲートウェイに通します。

## 参照エコシステム

TraceMind は React、Vite、Express、Prisma、Playwright、Vitest、PyMuPDF、arXiv、OpenAlex、Crossref、Semantic Scholar、Zotero などのオープンな技術と学術インフラを利用または参照しています。
