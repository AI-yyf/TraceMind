# TraceMind

TraceMind는 진지한 연구 읽기와 조사를 위한 AI 연구 워크벤치입니다. 논문 발견, 증거 추출, 연구 노드, 판단 작성, 근거 기반 대화를 하나의 추적 가능한 흐름으로 연결합니다.

## 해결하는 문제

- 연구 자료가 논문 데이터베이스, PDF, 메모, 채팅 기록에 흩어져 있습니다.
- AI 답변은 자연스럽지만 근거 경로가 숨겨지기 쉽습니다.
- 연구 주제는 계속 변하지만 일반 채팅 도구에는 장기적인 주제 기억이 부족합니다.

TraceMind는 논문, 그림, 수식, 인용, 노드, 대화를 같은 맥락에 두어 각 주장 뒤의 증거를 확인할 수 있게 합니다.

## 사용 방법

1. 백엔드와 프런트엔드를 실행합니다.
2. Settings에서 언어 모델과 선택적 비전 모델을 설정합니다.
3. 연구 주제를 만들거나 엽니다.
4. 논문 탐색을 실행하고 후보 논문을 검토합니다.
5. 노드 페이지에서 증거, 그림, 수식, 인용을 확인합니다.
6. 워크벤치에서 근거 있는 질문을 이어가고 결과물을 내보냅니다.

## 작동 방식

TraceMind는 React + Vite 프런트엔드, Express + Prisma 백엔드, Omni 모델 게이트웨이, 생성 데이터 스냅샷으로 구성됩니다. 백엔드는 학술 검색원을 통합하고 PDF 증거를 추출하며 주제 노드를 만들고 모델 호출을 설정 가능한 게이트웨이로 라우팅합니다.

## 참고 생태계

TraceMind는 React, Vite, Express, Prisma, Playwright, Vitest, PyMuPDF, arXiv, OpenAlex, Crossref, Semantic Scholar, Zotero 등의 오픈 기술과 학술 인프라를 사용하거나 참고합니다.
