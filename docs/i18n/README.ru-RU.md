# TraceMind

TraceMind — это персональная исследовательская рабочая среда с ИИ для серьезного чтения и анализа. Она связывает поиск статей, извлечение доказательств, исследовательские узлы, формирование выводов и уточняющие вопросы в один прослеживаемый процесс.

Это не просто чатбот и не список статей, а долгосрочное рабочее пространство для развития личной исследовательской темы.

## Какую Проблему Решает

- Исследовательские материалы разбросаны между базами статей, PDF, заметками и историей чатов.
- Ответы ИИ могут звучать убедительно, но скрывать путь к доказательствам.
- Исследовательские темы развиваются, а обычные чаты не имеют долговременной памяти темы.

TraceMind держит статьи, рисунки, формулы, цитаты, узлы и диалоги в одном контексте, чтобы исследователь мог проверить доказательства за каждым утверждением.

## Как Использовать

1. Запустите backend и frontend.
2. Настройте языковую модель и, при необходимости, визуальную модель в Settings.
3. Создайте или откройте исследовательскую тему.
4. Запустите поиск статей и просмотрите кандидатов.
5. Читайте страницы узлов с доказательствами, рисунками, формулами и цитатами.
6. Продолжайте контекстные вопросы в workbench и экспортируйте результаты.

## Как Это Работает

TraceMind объединяет frontend на React + Vite, backend на Express + Prisma, шлюз моделей Omni и подготовленные generated data. Backend агрегирует академические источники, извлекает доказательства из PDF, строит тематические узлы и направляет вызовы моделей через настраиваемый шлюз.

## Экосистема

TraceMind использует или учитывает React, Vite, Express, Prisma, Playwright, Vitest, PyMuPDF, arXiv, OpenAlex, Crossref, Semantic Scholar, Zotero и связанную открытую академическую инфраструктуру.
