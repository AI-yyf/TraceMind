# 溯知 TraceMind

溯知是一个面向严肃研究工作的 AI 研究工作台。它把论文发现、证据抽取、研究节点、判断生成和追问对话组织成同一个可追溯的闭环。

## 它解决什么问题

- 研究材料散落在论文库、PDF、笔记和聊天记录里。
- AI 输出常常很流畅，却很难追问证据来自哪里。
- 研究主题会持续演进，但普通聊天工具缺少长期主题记忆。

溯知把论文、图表、公式、引用、节点和对话放在同一个上下文里，让研究者能看见结论背后的证据链。

## 如何使用

1. 启动后端和前端。
2. 在设置页配置语言模型和可选视觉模型。
3. 创建或打开研究主题。
4. 运行论文发现，筛选候选论文。
5. 阅读节点页，检查证据、图表、公式和引用。
6. 在工作台里继续追问、整理判断并导出成果。

## 工作原理

溯知由 React + Vite 前端、Express + Prisma 后端、Omni 模型网关和生成数据快照组成。后端聚合学术搜索源，抽取 PDF 证据，构建主题节点，再通过模型网关生成可审计的研究内容。

## 参考生态

溯知使用或参考 React、Vite、Express、Prisma、Playwright、Vitest、PyMuPDF、arXiv、OpenAlex、Crossref、Semantic Scholar 和 Zotero 等开源项目与开放学术接口。
