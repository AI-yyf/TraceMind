TraceMind исходит из простой реальности: одного обновления в исследованиях почти никогда не хватает, чтобы увидеть весь ход развития направления.

Современные исследования в области ИИ быстры, шумны и сильно зависят от трендов. Резюме появляются мгновенно, но настоящее понимание накапливается гораздо медленнее. TraceMind задаёт другой вопрос: может ли ИИ отслеживать литературу, накапливать доказательства и отвечать, опираясь на них, чтобы стать верным и строгим исследовательским помощником?

## О проекте

TraceMind — это персональный исследовательский AI-воркбенч. Он подходит студентам, независимым исследователям, инженерам, техническим лидерам и аналитикам, которым нужно превратить большое количество статей в целостную картину.

| Типичная проблема | Чем помогает TraceMind |
| --- | --- |
| Слишком много статей и нет основной линии | карты тем, графы узлов, ключевые статьи и реальный прогресс |
| Ответы ИИ звучат гладко, но плохо обоснованы | ответы, связанные со статьями, PDF, рисунками, формулами и цитатами |
| Хорошие вопросы теряются в чатах и заметках | тематическое рабочее пространство с памятью и экспортом |
| Много следования хайпу, мало накопления | темы, растущие из реального материала |

## Зачем он нужен

Исследования часто буксуют не из-за отсутствия информации, а потому что понимание не успевает накопиться.

Обычные чат-инструменты хорошо отвечают, но плохо сохраняют:
- почему был сделан вывод
- какие доказательства его поддерживают
- что остаётся неопределённым
- как направление меняется со временем

TraceMind строится вокруг четырёх принципов:
- `сначала доказательства`
- `сначала память`
- `сначала структура`
- `человек остаётся носителем финального суждения`

## Сильные стороны

- `Страницы тем показывают реальный ход исследования`, а не искусственную фазу планирования.
- `Страницы узлов — это структурированные исследовательские представления` с ключевым вопросом, важными статьями, цепочкой доказательств, методами, выводами, ограничениями и итоговым суждением.
- `Доказательства остаются рядом` с финальным ответом.
- `Последующие вопросы не теряют контекст` темы и узла.
- `Система рассчитана на self-hosting` и контроль над моделями, ключами и данными.

## Быстрый старт

Требования:
- Node.js `18+`
- npm `9+`
- Python `3.10+`
- API-ключ хотя бы одного поставщика моделей

Backend:

```bash
cd skills-backend
npm install
cp .env.example .env
npm run db:generate
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Стандартные адреса:
- frontend: `http://localhost:5173`
- backend health: `http://localhost:3303/health`

Docker:

```bash
docker compose up --build
```

## Первые 15 минут

1. Запустите backend и frontend.
2. Настройте хотя бы одного поставщика моделей.
3. Создайте тему, которую действительно хотите отслеживать долгое время.
4. Запустите поиск статей и просмотрите кандидатов.
5. Оставьте только те работы, которые действительно относятся к основной линии темы.
6. Откройте исследовательский вид узла и сначала прочитайте структурированный обзор.
7. Задайте проверочный вопрос, например: `Какое доказательство в этой ветке самое слабое?`
8. Экспортируйте результат или продолжайте наращивать тему.

## Как устроен процесс

TraceMind организует исследование как цикл:
- поиск статей
- отбор и принятие кандидатов
- извлечение доказательств из PDF
- построение исследовательских узлов
- формирование поэтапных суждений
- контекстные уточняющие вопросы
- экспорт заметок и отчётов
- возврат всего в память темы

## Сравнение

| Инструмент | В чём силён | Роль TraceMind |
| --- | --- | --- |
| Zotero | сбор, аннотации, цитирование | превращает литературу в узлы, цепочки доказательств и суждения |
| NotebookLM | вопросы по заданному набору источников | удерживает эти вопросы внутри долгоживущей темы |
| Elicit | поиск и обзорные процессы | сильнее ориентирован на длительное личное исследование |
| Perplexity | быстрые ответы с источниками | превращает разовый ответ в память темы |
| Obsidian / Notion | личные заметки и организация | добавляет отслеживание литературы и grounded AI |
| ChatGPT / Claude | рассуждение и письмо | даёт модели исследовательскую комнату вместо пустого чата |

## Для кого подходит

TraceMind особенно полезен, если вы:
- следите за исследовательским направлением неделями и месяцами
- хотите сравнивать статьи, а не просто собирать их
- пишете обзоры, технические записки или исследовательские брифы
- хотите сами контролировать данные и настройки моделей

Он менее подходит, если вам нужен только:
- разовый факт
- красивый ответ без возврата к доказательствам
- универсальный корпоративный knowledge base

## Завершение

Трудно увидеть реальное направление исследований по одному обновлению. Ещё труднее, когда экосистема вознаграждает скорость, следование трендам и поверхностную новизну.

TraceMind пытается сделать ИИ помощником, который отслеживает литературу, накапливает доказательства и поддерживает дальнейшие вопросы на этой основе. Не голосом громче самой науки, а инструментом, который помогает яснее увидеть её форму.
